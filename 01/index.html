RO.ME "3 DREAMS OF BLACK"

RO.ME is a interactive music video in which the viewer can modify the music video by moving around the mouse cursor to invoke different events. In the beginning you "create" a path of animals via the cursor and at the end you "fly" by moving as a flock of birds to explore the generated world.

Technology: The RO.ME music video uses HTML and Javascript (Three.js library) code with WebGL to render the 3D graphics. The assests were created in Blender. WebGL uses hardware accleration which means that it uses the GPU of the client to generate the graphics reducing potential load on the server.
